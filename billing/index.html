הדבק את קוד ה-Firebase Config כדי לחבר את הנתונים לענן

❓ איך מקבלים Firebase Config?
  1. כנס ל-firebase.google.com והתחבר עם חשבון Google
  2. לחץ Add project ← תן שם ← המשך
  3. בתפריט השמאלי ← ⚙️ Project settings
  4. תחת Your apps ← לחץ </> (Web) ← Register app
  5. העתק את אובייקט ה-firebaseConfig והדבק למעלה
  6. חזור לפאנל ← Build → Firestore Database ← Create database ← בחר Start in test mode
